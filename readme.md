# Jófogás|Vatera étterem kalauz 🍴

Céges étterem ajánló app, OpenStreetMap térképpel és Supabase backenddel.

## Lokális futtatás

1. Klónozd a repót: `git clone <repo-url>`
2. Másold át a `config.example.js`-t `config.js`-re
3. Töltsd ki a Supabase adatokkal
4. Nyisd meg az `index.html`-t a böngészőben

## Deploy

Netlify-on automatikusan deploy-olódik a `main` branch push-ára.
A Supabase kulcsok a Netlify environment változókban vannak tárolva.
